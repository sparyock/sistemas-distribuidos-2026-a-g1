# Lineamientos para el MVP

## Documentación
- Mantener documentación clara del proyecto.
- Aplicar buenas prácticas de desarrollo.

## Ambientes de desarrollo
Simular los siguientes ambientes:

- DEV
- QA
- MAIN (release)

## Evidencias requeridas
- Entregar evidencia de certificación o manejo de GitHub.

## Taller individual
Cada estudiante deberá construir de manera individual un **taller completo** basado en un **carrito de compras**, que incluya:

- Crear usuario
- Crear productos
- Crear factura
- Crear detalle de factura

Cada estudiante deberá **simular los ambientes DEV → QA → MAIN**, siguiendo el manual entregado.

## Reflexión del proyecto
Durante la presentación o cierre del ejercicio se debe responder:

- ¿Qué salió bien durante el desarrollo?
- ¿Qué dificultades se presentaron y cómo se solucionaron?
- ¿Qué aprendizajes quedaron del proceso?

## Weekly (seguimiento semanal)
Construir un registro semanal que incluya:

- Qué se hizo
- Qué no se logró completar
- Qué se realizará en la siguiente semana

## Participación
Evaluar:

- Porcentaje de apoyo de cada integrante dentro del equipo.

## Acciones por parte del docente
- Proveer más material de apoyo.
- Definir con mayor claridad:
  - Actividad
  - Tiempo de entrega
  - Relación con las Historias de Usuario (HU)

# ADR – Arquitectura de Software

Cada integrante deberá aplicar **5 ADR (Architecture Decision Records)** dentro del proyecto.

Los ADR pueden estar relacionados con:

- Patrones de diseño
- Patrones estructurales
- Patrones comportamentales

## Proceso para los ADR
1. Identificación de patrones  
2. Análisis del proyecto para determinar qué ADR pueden aplicarse  
3. Identificación de antipatrones  
4. Detección de segmentos de código con malas prácticas  
5. Diseño e implementación del ADR  
6. Análisis del impacto del ADR sobre el sistema  

## Weekly de ADR (Semana 17)
- Listar todos los ADR posibles del proyecto.
- De esa lista, cada integrante debe implementar **3 ADR**.
- Entregar un **informe de impacto**, explicando cómo cada ADR mejora el sistema.
